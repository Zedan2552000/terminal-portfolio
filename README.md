# 💻 Interactive Terminal Portfolio

A retro, cyberpunk-style interactive developer portfolio built purely with HTML, CSS, and Vanilla JavaScript. 

This project simulates a command-line interface, allowing visitors to discover your skills, projects, and contact information by typing Linux-style commands.

## ✨ Features
- **Authentic Terminal Feel:** Emulated typing, blinking cursor, and command history.
- **Pure Frontend:** No framework overhead. Just fast, clean HTML/CSS/JS.
- **Customizable Commands:** Easily add your own commands in `script.js`.
- **Responsive Design:** Works perfectly on desktop and mobile browsers.
- **Glassmorphism Hints:** Subtle shadows and glowing neon effects for a premium look.

## 🚀 Usage (Commands)
Type the following commands in the terminal:
- `help` - Show all available commands
- `about` - Learn more about me
- `skills` - View my technical skills
- `projects` - See what I've been working on
- `github` - Opens my GitHub profile
- `theme` - Toggle light/dark mode
- `clear` - Clear the terminal screen

## 🛠️ How to Customize
1. Open `script.js`
2. Locate the `ABOUT`, `SKILLS`, and `PROJECTS` constants.
3. Replace the placeholder text with your own information.
4. Add new commands to the `commands` object.
